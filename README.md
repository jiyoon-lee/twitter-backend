## 이 프로젝트를 통해 무엇을 얻고 싶은가

## 사용한 기술

- bcrypt
  비밀번호를 암호화해서 저장하기 위해 사용

- cors
  브라우저를 통해 프런트와 통신하기 위해
- dotenv
  공개되어서는 안되는 키 또는 환경에 따라 달라지는 값을 보관하기 위해 사용

- express

  - helmet
  - morgan
  - express-async-errors
  - express-validator

- jsonwebtoken
  토큰 발급은 위해 사용

- uuid
  tweet의 uniqueId 저장을 위해 사용

- mysql2
- socket.io
